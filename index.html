Support & FAQ

Frequently asked questions

Quick answers to the most common questions about our products and services.

What is website hosting?

Website hosting is a service that allows individuals and organizations to make their website accessible on the internet by storing it on a server.

What types of website hosting are available?

Options include shared hosting, VPS hosting, dedicated hosting, cloud hosting and managed hosting — each catering to different needs and performance requirements.

What is shared hosting?

Shared hosting is where multiple websites share the same server resources, making it a cost-effective option for smaller websites.

What is reseller hosting?

Reseller hosting lets individuals or companies purchase hosting resources and resell them to their clients, managing multiple websites under one account.

What is VPS hosting?

VPS hosting provides a virtual private server with dedicated resources and increased control — ideal for growing websites needing more power and flexibility.

What is dedicated hosting?

Dedicated hosting provides an entire server exclusively for one client, offering maximum performance, control and security for high-traffic websites.

What is cloud hosting?

Cloud hosting uses a network of virtual servers to host websites, providing scalable resources, high availability and enhanced performance.

What is managed hosting?

Managed hosting includes comprehensive support where the provider handles server maintenance, security and backups, so you can focus on your website.
